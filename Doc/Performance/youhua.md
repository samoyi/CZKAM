# 优化策略
* 静态资源托管：图片、脚本、视频
* 外部脚本等（如Vue.js）使用国内公共CDN
* 分步加载
* 预判加载
* 一次请求加载完所有的数据
* 缓存耗时操作
* 脚本合并压缩
