# 优化策略
* 静态资源托管：图片、脚本、视频
* 分步加载
* 预判加载
* 一次加载完所有的数据
* 缓存耗时操作
* 脚本合并压缩

## Preloading
* 首页加载完成后，预加载各个分页首个版块的图片

## Lazy Loading
* 分页首个版块加载完成后，加载其他版块的图片
